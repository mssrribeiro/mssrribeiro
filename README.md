# Portfólio - Mayara Ribeiro

## Como usar
1. Abra `index.html` no navegador (duplo clique).
2. Edite os textos no `index.html` e ajuste o visual em `styles.css`.
3. Para publicar (grátis), sugestões:
   - GitHub Pages (repo público)
   - Netlify (arrastar e soltar a pasta)
   - Vercel

## Onde editar rápido
- Nome, títulos, contatos e links: `index.html`
- Cores, layout e espaçamentos: `styles.css`
- Pequenos scripts (ano/efeitos): `script.js`

## Observação
A imagem `img/profile.jpeg` é um placeholder. Você pode substituir por `profile.jpg` e trocar no HTML.
